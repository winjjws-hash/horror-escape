# 백룸: 열쇠와 그림자

WASD로 움직이며 열쇠를 찾고, 움직이는 귀신을 피해 출구까지 도망치는 백룸 생존 게임입니다. 브라우저에서 바로 실행할 수 있고, GitHub Pages에 올려도 별도 빌드 없이 동작합니다.

## 플레이

- 이동: `W`, `A`, `S`, `D` 또는 방향키
- 목표: 스테이지마다 열쇠를 모두 먹고 녹색 출구로 이동
- 목숨: 3개. 귀신과 닿으면 목숨 1개, 시간, 공포 수치가 함께 깎입니다.
- 지옥: 저주 괴물이 추가로 등장합니다. 닿으면 시야가 좁아지고 이미 얻은 열쇠 하나를 잃을 수 있습니다.
- 패배: 목숨 0, 공포 100%, 시간 초과

## 스테이지

1. Level 0: 넓어진 노란 복도
2. Level 1: 전등이 꺼진 병동
3. Level 2: 출구가 움직이는 방

## 실행

`index.html` 파일을 브라우저로 열면 바로 시작됩니다.

## GitHub Pages 배포

1. GitHub 계정 `winjjws-hash`의 저장소에 파일을 올립니다.
2. 저장소의 `Settings > Pages`에서 `Deploy from a branch`를 선택합니다.
3. `main` 브랜치와 `/root` 폴더를 선택하고 저장합니다.

배포 후 주소 예시는 아래와 같습니다.

```text
https://winjjws-hash.github.io/horror-escape/
```

## 인코딩

모든 소스 파일은 UTF-8 기준으로 작성했습니다. `index.html`에는 `<meta charset="utf-8">`이 들어 있고, `.editorconfig`에서도 문자셋을 UTF-8로 고정합니다.
