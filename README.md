# 폐병동 13호

한글 UI와 저장 기능이 들어간 정적 웹 방탈출 게임입니다. 브라우저에서 바로 실행할 수 있고, GitHub Pages에 올려도 별도 빌드 없이 동작합니다.

## 실행

`index.html` 파일을 브라우저로 열면 바로 시작됩니다.

## GitHub Pages 배포

1. GitHub 계정 `winjjws-hash`에 새 저장소를 만듭니다.
2. 이 폴더의 파일을 저장소에 올립니다.
3. 저장소의 `Settings > Pages`에서 `Deploy from a branch`를 선택합니다.
4. `main` 브랜치와 `/root` 폴더를 선택하고 저장합니다.

배포 후 주소 예시는 아래와 같습니다.

```text
https://winjjws-hash.github.io/저장소이름/
```

## 인코딩

모든 소스 파일은 UTF-8 기준으로 작성했습니다. `index.html`에는 `<meta charset="utf-8">`이 들어 있고, `.editorconfig`에서도 문자셋을 UTF-8로 고정합니다.
